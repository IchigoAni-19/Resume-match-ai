# Resume-Match-AI

An intelligent, full-stack web application designed to bridge the gap between candidate profiles and job requirements. By leveraging Generative AI, this platform analyzes a user's resume and target job description to dynamically generate personalized interview preparation reports and tailored PDF resumes.

## 🚀 Features
* **AI-Driven Interview Strategy:** Generates behavioral questions, technical questions, and daily preparation roadmaps based on skill gaps.
* **Smart Resume Generator:** Automatically crafts a targeted, ATS-friendly PDF resume using Puppeteer and Gemini AI.
* **Secure Authentication:** Robust JWT-based authentication system with encrypted passwords and HTTP-only cookies.
* **Modern Architecture:** Built following a strict 4-layer frontend architecture (UI, Hook, State, API) for maximum scalability and clean code.

## 🛠️ Tech Stack
**Frontend:**
* React.js (Vite)
* SCSS (Custom styling architecture)
* React Router DOM
* Axios

**Backend:**
* Node.js & Express.js
* MongoDB & Mongoose
* Google Gemini AI SDK (`@google/genai`)
* Puppeteer (for PDF generation)
* JWT & Bcrypt.js
* Multer (File handling)
